
Reviewer cheat sheet
Category	What it includes	What to capture	Example	Common issue to flag
Percentages and counts	Raw counts and proportions	Number of patients/events and any % tied to them	“18/120 patients (15%) had rash”	% without denominator, count without context, mismatch between count and %
Totals and subgroup counts	Overall N and subgroup Ns	Total population, arm sizes, demographic or clinical subgroups	“N=240; 120 treatment, 120 placebo; 64 age ≥65”	Subgroup totals do not add up to overall total
Treatment arm consistency	Alignment of arm labels and sample sizes across sections	Arm name, arm N, outcome tied to correct arm	“Drug A (n=120), placebo (n=120)”	Same arm reported with different N later, or result assigned to wrong arm
Time references	Timing, duration, follow-up, baseline references	Day/week/month, duration, time window, follow-up period	“at Week 24,” “within 30 days,” “median follow-up 11.2 months”	Missing time anchor, inconsistent visit/timepoint naming
Statistical values	Inferential and summary statistics	p-values, CIs, HR, OR, RR, mean, median, SD, IQR	“HR 0.72; 95% CI 0.58–0.90; p=0.004”	CI inconsistent with point estimate, impossible p-value formatting, unsupported claim of significance
Units and dosing values	Measurement units and treatment regimen details	mg, mg/kg, mL, mmol/L, frequency, route, infusion duration	“200 mg IV every 3 weeks”	Missing unit, wrong conversion, dose frequency omitted

Quick annotation guidance
1. Percentages and counts
Use this label when the text gives a direct number of people, events, or outcomes, with or without a percentage.
Examples:
“27 patients withdrew.”
“12% experienced headache.”
“9 of 75 subjects (12%) had serious infection.”
Flag when:
9/75 is reported as 14%
“12% improved” but no denominator is available
2. Totals and subgroup counts
Use this when the text defines the full analysis population or any subgroup size.
Examples:
“A total of 310 patients were screened.”
“The safety population included 202 patients.”
“Among them, 88 had prior biologic exposure.”
Flag when:
treatment-arm counts sum to more than total N
subgroup percentages imply the wrong base population
3. Treatment arm consistency
Use this when checking whether the same study arm is described consistently across methods, results, tables, and figures.
Examples:
“Low-dose (n=80), high-dose (n=82), placebo (n=81)”
Later: “Response in high-dose was 41/82”
Flag when:
high-dose is later shown as n=84 without explanation
placebo result is reported under active treatment label
randomization ratio and actual Ns do not fit
4. Time references
Use this for any explicit or implied timing attached to assessments, events, endpoints, exposure, or follow-up.
Examples:
“By Day 14”
“through Week 52”
“median time to response was 17 days”
“change from baseline to Month 6”
Flag when:
endpoint timing differs across sections
“overall survival was 80%” but no timepoint is given
baseline and post-baseline windows are mixed
5. Statistical values
Use this for all numerical evidence supporting claims.
Examples:
“p=0.03”
“95% CI 1.2–3.8”
“mean 7.4 (SD 1.2)”
“median PFS 9.8 months”
Flag when:
claim says “significant” but p>0.05
CI excludes neither null nor supports stated conclusion
median is paired with SD instead of a more fitting spread measure unless justified
6. Units and dosing values
Use this when a number is tied to a scientific, clinical, or dosing unit.
Examples:
“5 mg/kg”
“0.9% saline”
“ALT 54 U/L”
“10 mL orally twice daily”
Flag when:
number is present but unit is missing
“mg” used where “mg/kg” is expected
route/frequency omitted from treatment description
7. Derived metrics
Use this for calculated outcomes rather than raw observations.
Examples:
“Objective response rate was 28%.”
“Relative risk reduction was 35%.”
“Annualized relapse rate was 0.21.”
“Mean percent change from baseline was −14.6%.”
Flag when:
derived metric cannot be reconciled with raw counts
formula basis is unclear
change from baseline is reported but baseline/follow-up values do not support it
One worked example
Sentence:
“Among 180 randomized patients, 90 received Drug X and 90 placebo. At Week 12, 27 Drug X patients (30%) and 12 placebo patients (13.3%) achieved response; risk ratio 2.25 (95% CI 1.22–4.14), p=0.009. Drug X was dosed at 150 mg once daily.”
How to annotate:
Totals and subgroup counts: 180 total, 90 Drug X, 90 placebo
Treatment arm consistency: Drug X and placebo arms clearly defined
Time references: Week 12
Percentages and counts: 27 patients (30%), 12 patients (13.3%)
Derived metrics: achieved response rate, risk ratio
Statistical values: 95% CI, p=0.009
Units and dosing values: 150 mg once daily
Fast decision rule
A simple way to separate them:
If it is a raw number or proportion, use percentages and counts
If it is the size of the whole group or subgroup, use totals and subgroup counts
If it checks whether arms stay aligned, use treatment arm consistency
If it tells when, use time references
If it supports inference or uncertainty, use statistical values
If it includes measurement or regimen units, use units and dosing values
If it is calculated from other numbers, use derived metrics
